## 📥 Comment télécharger et installer ?

Comme cette extension est un outil privé (non-disponible sur le Chrome Web Store), l'installation se fait manuellement en **Mode Développeur**.

1.  **Récupérer les fichiers** :
    *   Clique sur le bouton vert **`Code`** en haut de cette page, puis sur **`Download ZIP`**.
    *   Extrais le contenu du fichier `.zip` dans un dossier sur ton bureau (nomme-le `Nextera-Bot` par exemple).
2.  **Ouvrir les Extensions** :
    *   Dans ton navigateur (Chrome, Brave, Edge ou Opera), tape `chrome://extensions/` dans la barre d'adresse.
3.  **Activer le Mode Développeur** :
    *   Bascule l'interrupteur **"Mode développeur"** en haut à droite sur **ON**.
4.  **Charger le Bot** :
    *   Clique sur le bouton **"Charger l'extension décompressée"** (Load unpacked).
    *   Sélectionne le dossier où tu as extrait les fichiers (celui qui contient le fichier `manifest.json`).
5.  **Épingler l'icône** :
    *   Clique sur l'icône "Puzzle" en haut à droite de ton navigateur et épingle **NEXTERA BOT**.

---

## 🚀 Action à réaliser pour que ça marche

L'extension utilise le **"Sniffing"** : elle doit voir passer une action réelle pour apprendre à la reproduire à l'infini.

1.  **Ouvre Discord** : Connecte-toi sur ton compte dans ton navigateur.
2.  **Vérifie le Widget** : Un carré rouge `NEXTERA BOT` apparaît en haut à droite avec le message `ACTION REQUIRED ON DISCORD`.
3.  **L'Action "Mère"** :
    *   Lance l'activité **Last Meadow Online** dans un salon.
    *   Lance une récolte (clique sur **Start**).
    *   Attends la fin du compte à rebours dans le jeu.
    *   Clique sur le bouton pour valider la récolte (**Complete/Finish**).
4.  **Validation** : 
    *   Dès que tu cliques sur valider, le widget passe instantanément au **VERT** 🟢.
    *   Le bot a maintenant "appris" la requête et commence à la répéter tout seul en boucle.
    *   **Tu peux maintenant réduire la fenêtre ou changer d'onglet**, le bot travaille pour toi.

## 💡 Infos utiles & Astuces

*   **🕹️ Personnalisation** : Clique sur le bandeau bleu du widget pour le **déplacer** où tu veux sur ton écran. Utilise le bouton **[-]** pour le rendre compact et ne pas gêner ta navigation.
*   **💎 Suivi du Loot** : Le widget affiche en temps réel les ressources cumulées (Bois, Métal, XP, etc.) depuis le lancement du bot.
*   **⚠️ Gestion des Erreurs** :
    *   **Point JAUNE / Message "Retry"** : Tu as été "Rate Limited" par Discord (trop de requêtes). Pas de panique, le bot attend 5 secondes et reprend tout seul.
    *   **Point ROUGE** : Ta session a expiré ou tu as rafraîchi la page. Refais simplement l'étape **"Action à réaliser"** pour relancer.
*   **🌍 Mode Partout** : Via la petite icône de l'extension en haut du navigateur, tu peux choisir d'afficher le widget sur tous les sites web ou uniquement sur Discord.
